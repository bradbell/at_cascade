node_split_table

Node Split Table

  • node_split_id

  • node_id

This table specifies the nodes at which the cascade will be split by the value of the splitting covariate. If this table is not empty, the split_covariate_name must appear and the Split Reference Table must not be empty.

node_split_id

is the Primary Key for this table.

node_id

This column has type integer and contains a dismod_at node_id at which the analysis will be split by the splitting covariate; i.e., this node will be fit both before and after the splitting. The set of values in this column determine the node_split_set.

xsrst input file: xsrst/all_node_db.xsrst
