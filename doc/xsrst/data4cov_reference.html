data4cov_reference¶

}

Use Data Table Average Covariates for Reference¶

  • Syntax

  • root_node_database

    • option Table

    • data Table

    • covariate Table

  • all_node_database

    • all_option Table

  • trace_interval

    • all_cov_reference Table

Syntax¶

# at_cascade.data4cov_reference
    all_node_database  = None,
    root_node_database = None,
    trace_interval     = None,
# )

root_node_database¶

This argument can’t be None and is the root_node_database. Only the following tables in this database are used: option, data, node, covariate and only the covariate table is modified.

option Table¶

This table must have a row with option_value equal to parent_node_name. Only this row of the option table is used.

data Table¶

Only the following columns of this table are used: data_id, node_id, x_0, … x_n, where n is the number of covariates minus one.

covariate Table¶

The covariate reference value for the relative covariates is changed to agree with the new value in the all_cov_reference table (described below) for the node_id and split_reference_id corresponding to the root_node_database.

all_node_database¶

This argument can’t be None and is the :ref:all_node_db`. Only the All Node Covariate Reference Table and All Node Option Table are used.

all_option Table¶

The split_covariate_name and absolute_covariates rows of this table (if they exist) are the only rows of this table that are used.

trace_interval¶

If trace_interval is None no tracking is printed. Otherwise, a message will once per trace_interval nodes to indicate the progress of this processing (because it can take a while).

all_cov_reference Table¶

The reference created using the average of the covariates in the data table. (The average is rounded off to and unspecified number of digits. This makes it easier to set reference values in the root_node_database to be the same.)

  1. If there is an all_cov_reference table on input, all its information is lost.

  2. The node_id for the data table row is the node_id for the new all_cov_reference row, or a descendant of all_cov_reference node.

  3. The difference of all the covariate values in the data table row from the corresponding reference in the covariate table is less than or equal the maximum difference for that covariate.

  4. If there is no row of the data table that satisfies conditions 1 and 2 for this all_cov_reference row, the resulting reference value is zero.
