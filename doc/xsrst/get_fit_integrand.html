get_fit_integrand¶

Determine the Set of Integrands in Data Table¶

  • Syntax

  • fit_node_database

  • fit_integrand

Syntax¶

# data_integrand = at_cascade.get_fit_integrand(
    fit_node_database = None ,
# )

fit_node_database¶

is a python string specifying the location of a fit_node_database relative to the current working directory. Note that only the data table in this database is used and it is the same data table as the data table in the root_node_database. This argument can’t be None.

fit_integrand¶

The return value fit_integrand is a python set of integrand_id that appear in the data table in the fit_node_database. Furthermore there is a row in the data table where each such integrand_id is not held out.
