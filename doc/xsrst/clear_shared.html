clear_shared

Clear at_cascade Shared Memory

  • Syntax

  • Purpose

Syntax

clear_shared()

Purpose

This routine clears the at_cascade memory pointers that are in use. For example, you may get the following error message when trying to run the cascade:

FileExistsError: [Errno 17] File exists: ‘/at_cascade_number_cpu_inuse’

This may happen if the previous Run With Specified Maximum Number of Processes did not terminate cleanly; e.g., if the system crashed.
