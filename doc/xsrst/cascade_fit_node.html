Syntax¶

# at_cascade.cascade_fit_node(
    all_node_database = None,
    fit_node_database = None,
    node_table        = None,
# )

all_node_database¶

is a python string specifying the location of the all node database relative to the current working directory.

all_node_dir¶

is the directory where the all node database is located.

fit_node_database¶

is a python string specifying the location of the fit_node_database relative to all_node_dir. This must be a valid root_node_database.

fit_node¶

The fit_node_database must be fit_node/dismod.db, or end with the text /fit_node/dismod.db where fit_node is the name of the fit_node for this database.

fit_node_dir¶

is the directory where the fit node database is located.

node_table¶

This is a python list where node_table*[*node_id] is a python dictionary representation of the corresponding row of the dismod_at node table. This node table is the same as the node table in fit_node_database. (It is the same for all the fits and this avoids reading it each time.)

Results¶

The results for this fit are in files in the fit_node_dir directory. In addition, for each child node of fit_node_name are in a sub-directory called the child node name.
