mulcov_freeze_table

Mulcov Freeze Table

  • mulcov_freeze_id

  • fit_node_id

  • split_reference_id

  • mulcov_id

This table specifies the jobs at which the cascade will freeze the value of covariate multipliers for its descendants (starting with its child jobs). To be specific, the lower and upper limit for the covariate multiplier in descendants of the job will be optimal value determined by job; see parent_job_id in the job table.

mulcov_freeze_id

is the Primary Key for this table.

fit_node_id

is the fit_node_id for the job that will freeze the multiplier.

split_reference_id

is the split_reference_id for the job that will freeze the multiplier. This value is null if and only if the split_reference_table is empty.

mulcov_id

is the primary key in the dismod_at mulcov table for the multiplier that is frozen.
