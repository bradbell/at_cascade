all_node_db¶

The All Node DataBase¶

  • Primary Key

This data base contains information for all the nodes between the root_node and the fit_goal_set.

Primary Key¶

If tbl_name is the name of a table in this database, the primary key column has name tbl_name_id . This column has type integer , its initial value is zero, and it increments by one for each row.
