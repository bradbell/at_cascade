get_parent_node

Get Parent Node Name From Option Table

  • Syntax

  • database

  • parent_node_name

Syntax

# parent_node_name = at_cascade.get_parent_node(
    database = None
# )

database

This is a dismod_at database.

parent_node_name

The return value parent_node_name is the database option table option_value for the row where option_name is parent_node_name. An assert will be generated if no such option_name is found in the option table.
