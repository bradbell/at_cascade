all_option_table¶

All Node Option Table¶

  • Table Format

    • all_option_id

    • option_name

    • option_value

  • root_node_name

  • sex_level

  • in_parallel

This table specifies certain at_cascade options

Table Format¶

all_option_id¶

is the Primary Key for this table.

option_name¶

This column of the option table has type text. It specifies a name that is attached to each option value.

option_value¶

This column of the option table has type text. It specifies a value that is attached to each option name.

root_node_name¶

This option_name must appear and the corresponding option_value is the name of the root_node in the node table of the root_node_database . (The node table is the same in all the dismod_at databases.)

sex_level¶

If this option_name appears, the corresponding option_value is not yet specified. Otherwise, there will be no sex split in this cascade.

in_parallel¶

If this option_name appears, the corresponding option_value is not yet specified. Otherwise, the cascade will be run sequentially.
