option_name
===========
This column of the option table has type ``text``.
It specifies a name that is attached to each option value.

.. meta::
   :keywords: option_value

.. index:: option_value

.. _all_option_table.table_format.option_value:

option_value
============
This column of the option table has type ``text``.
It specifies a value that is attached to each option name.

.. meta::
   :keywords: root_node_name

.. index:: root_node_name

.. _all_option_table.root_node_name:

root_node_name
**************
This option_name must appear and the corresponding option_value
is the name of the :ref:`glossary.root_node` in the node table of the
:ref:`glossary.root_node_database` .
(The node table is the same in all the dismod_at databases.)

.. meta::
   :keywords: sex_level

.. index:: sex_level

.. _all_option_table.sex_level:

sex_level
*********
If this option_name appears, the corresponding option_value
is not yet specified.
Otherwise, there will be no sex split in this cascade.

.. meta::
   :keywords: in_parallel

.. index:: in_parallel

.. _all_option_table.in_parallel:

in_parallel
***********
If this option_name appears, the corresponding option_value
is not yet specified.
Otherwise, the cascade will be run sequentially.
